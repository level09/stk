# Commands

Use `uv` for Python commands.

Common commands:
- `uv run quart run`
- `uv run quart create-db`
- `uv run quart db current`
- `uv run quart db history`
- `uv run ruff check .`
- `uv run ruff format .`
- `uv run python checks.py`

Agent-operability commands:
- `uv run quart inspect routes --json`
- `uv run quart verify`
- `uv run quart verify --json`
- `uv run quart report`
